# Team Mechaholics - Premium Website

This is the completely rebuilt, premium React website for the Official Quad Bike Team of NIT Warangal, Team Mechaholics. 

## Features
- Built with React 19 + Vite 8
- Apple-style premium aesthetics using TailwindCSS v4 and glassmorphism.
- Hardware-accelerated animations via Framer Motion & GSAP.
- Dynamic scrolling and parallax effects.
- Multi-page layout with a dedicated Recruitment form (`/apply`).

## Getting Started

First, install dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) with your browser to see the result.
